# Ingredientes que inspiran

Landing page moderna para una marca de ingredientes naturales.

## Stack

- **Framework:** Next.js 15
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS
- **Iconos:** react-icons
- **Fuente:** Inter 

## API

`GET /api/products` — Devuelve array de productos (Limón, Tomate) con latencia simulada.
